# Loops & String
Loops are used to execute a piece of code again & again

Types of loops
1. while loop
2. do-while loop
3. for loop

>[!Note]

> for(let i=1; i<=5; i++) <br> {
> console.log("Hello world"); <br>
>}
>
>`while`(condition) <br> { <br>
